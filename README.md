# hamradio_tutorial_book
アマチュア無線入門時に、今時ゲーム等でも当たり前に実装されているように、一通りの体験をクリアしていく形式のチュートリアルがあったらいいよねという案。

## 本文データのライセンス
- クリエイティブコモンズ CC-BY-SA
  - CC :「著作権を守りつつ、一定のルールの範囲内で自由に作品を使っていいですよ」というライセンスです。
  - BY : 利用時は「作者の名前」を必ず表示する必要があります。
  - SA : 利用時は、その新作にも同じライセンスを付けて公開する必要があります。
- 自由にやっていいこと
  - このデータをアレンジ、応用してオリジナル作品を生み出す。
  - アレンジしたデータの作者名表記の次の行に自分の名前を追加する。
  - このデータや応用した作品を、同人誌やオンライン等で「配布」する。
  - このデータや応用した作品を、同人誌やオンライン等で「販売」する。
- やってはダメなこと
  - なりすまし、自作発言等、このアイディアやデータを自分が発明・作成したかのように偽ることは禁止です。
  - 既存の作者名を改変、削除する事は禁止です。
  - ライセンスは必ず「見える場所」に表示もしくは印刷する必要があります。変更、削除する事は禁止です。

【重要：ライセンスの適用範囲について】
- 本文（テキストデータ）には上記 CC BY-SA が適用する必要があります。
- ただし、「画像・ロゴ・装飾デザイン」については、必要に応じて「個別のライセンス」や「転載禁止・複製禁止」を設定する事ができます。
